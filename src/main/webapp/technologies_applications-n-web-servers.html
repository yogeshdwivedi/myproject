WebSphere
Websphere is also known as IBM Websphere. It is a brand of computer software that can be classified as application and integration middleware, and is an enterprise software. The main function or feature of IBM Websphere has been its ability to let users create applications and integrate them with other applications. IBM Websphere was first launched in 1998 as IBM Websphere Performance pack, and has gone through a number of interesting upgrades and development over the years.

Websphere is classified according to the abilities that are offered for different industries. Among the main products are the Websphere application server and the websphere application server family, Websphere eXtreme scale and IBM Workload Deployer.

Websphere Application server, also known as the WAS, is a software that is programmed to perform the functions of a web application server. It hosts Java based applications working as a middleware and a software framework. It is built using Open standards like Java EE, web services and XML. It is also supported across platforms like AIX, Linux, Windows, iOS and Solaris among other platforms. It also works under a number of web servers that include IBM HTTP Server, IIS, Apache HTTP server and others.

IIS
Internet Information Server now rechristened as Internet Information Services, is also popularly referred to as IIS. IIS is essentially a web server software application for use with Microsoft Windows due to its extension modules. The new release of the Internet Information Services supports NNTP, SMTP also other than FTPS, FTP, HTTPS and HTTP and is a part of the windows server family including Windows NT 4.0 and 2000. IIS manager is accessed through the Microsoft management console or Admin tools from the control panel. IIS is one of the top web servers across the world.

Internet information Server of IIS application might appear to be a series of HTML pages but actually made of a number of webclass containing webitems. Mostly designed and developed using Visual Basic. This VB application responds to requests from the browser and lives on the web server itself.

Though IIS applications seem to be very similar to DHTML applications and ASP applications, it is a completely independent and separate application. With IIS, it is possible to isolate multiple sites or individual web applications into IIS application pool or self contained process. This IIS application pool can actually communicate with the Operating System kernel directly.
